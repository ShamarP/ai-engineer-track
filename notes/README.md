# Notes

Running notes as I read. One file per source.

- `papers/` — paper-by-paper notes (filename: `{first-author}-{year}-{slug}.md`)
- `books/` — chapter-by-chapter notes (`aie-ch01.md`, `dmls-ch07.md`, etc.)

## Format

Lead with the question the source answers, then key claims, then your own take. Don't summarize — argue.

```
# {Title}

**Source:** {citation, link}
**Read on:** YYYY-MM-DD
**Why I read it:** {one line}

## The question
What problem is this trying to solve?

## The claim(s)
- bullet
- bullet

## My take
What surprised me? What do I disagree with? What's the next thing I want to know?
```
